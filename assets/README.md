- `favicon.svg` — favicon-ready SVG placeholder.
- `og-image.png` — 1200 × 630 social preview placeholder.

## Asset rules

- Keep SVGs optimized and accessible with `<title>` / `<desc>` when meaningful.
- Use PNG/JPG only for screenshots or rich previews.
- Do not include sensitive lead, partner, CRM, or financial data in screenshots.
- Use demo data only.
- Keep social preview images at 1200 × 630.
- Keep favicon SVG small and dependency-free.
