API boundary

The current repo includes example API docs and placeholder files only. A static host cannot process live POST requests unless a serverless/backend layer is added.

Do not describe example endpoints as live. Do not connect GPT actions, widgets, or forms to example files until production endpoints exist.

Recommended rate limits

  • Widget lead submissions: 10 per IP per hour until stronger anti-spam is added.
  • Partner stats reads: 60 per partner per minute.
  • Webhook endpoints: provider-specific limits plus signature verification.
  • Admin endpoints: authenticated operator-only and audit logged.
